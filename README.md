# 🚀 Convertify

Convertify is a simple and user-friendly web application that allows users to easily convert files between different formats.

## 🔧 Features

- 📄 **PDF to DOC** – Convert PDF files into editable Word documents  
- 📝 **DOC to PDF** – Convert Word documents into PDF format  
- 🖼️ **Images to PDF** – Combine images and convert them into a single PDF file  

## 💡 How It Works

1. Upload your file  
2. Choose the desired conversion format  
3. Click on convert  
4. Download the converted file instantly  

## 🎯 Purpose

The goal of Convertify is to provide a fast, reliable, and easy-to-use tool for everyday file conversion needs without complexity.

## 🛠️ Tech Stack

- Frontend: JavaScript 
- Backend: Python  
- Libraries: File handling and conversion libraries  

## 📌 Note

- Files are processed securely  
- No long-term storage of user files
